# Kiyo 🎵

A lightweight, modern local music player for Android (API 21+).

## Features
- Dynamic frosted-glass UI with deep-charcoal theme
- Poweramp-style multi-artist parsing engine with split-token & exception whitelist
- Discord Rich Presence (RPC) integration
- Last.fm scrobbling with offline cache & auto-flush
- Media3 / ExoPlayer backend with lifecycle-safe ForegroundService
- Full 32-bit + 64-bit ABI support

**App ID:** `com.space4414.kiyo`  
**Min SDK:** 21 (Android 5.0 Lollipop)  
**Target SDK:** 35
